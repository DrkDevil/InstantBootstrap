Getting Started

InstanstBootstrap makes it simple to create beautiful bootstrap themes using LESS, or SASS with just a few simple steps.

After the simple setup you will have all the files you need to build, and test a your bootstrap theme.


Simply download from our GitHub page, or within your terminal run the command: 

$ git clone https://github.com/DrkDevil/InstantBootstrap


The final step now is to navigate to the newly downloaded repo, and run 

$ npm install


Now that all of InstantBootstrap's dependencies have been downloaded, you are ready to start creating beautiful bootstrap themes.

No Configuration Needed

InstanstBootstrap has a simple folder structure and preset gruntfile, that makes it easy to jump right into developing.

InstantBootstrap/
├── source/ -- working files
│ ├── css/
│ ├── demo/
│ ├── js/
│ ├── sass/
│ └── less/
├── base_theme/ -- compiled files
│ ├── css/
│ ├── js/
│ └── fonts/
└── docs/ -- coming soon

Also included are all 17 of bootstraps original templates, so you can start laying out new sites imediately.

Easily Expandable

Use Bower to easily add new components, and expand the functionality of your themes. To keep with the stucture of InstantBootstrap a .bowerrc file has been added to direct all bower components to the source folder.
